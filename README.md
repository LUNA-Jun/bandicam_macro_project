# 반디카메라 스크린샷 매크로
-------------------------------

반디카메라로 스크린샷을 찍어야하는 일이 있었어 간단한 매크로를 만듬

-------------------------------
## mouse_position.py

마우스의 위치 좌표를 출력하고 ESC 누를시 종료
반디 카메라의 캡쳐 아이콘에 마우스를 올려두고 마우스 좌표 출력

-------------------------------
## bandicam_caputre.py

마우스 죄표를 넣고 ESC눌러 종료할 때 까지 캡쳐클릭을 계속 클릭
